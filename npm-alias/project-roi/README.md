# project-roi

Alias package. Running:

```bash
npx project-roi
```

launches **[token-roi](https://www.npmjs.com/package/token-roi)** — local-first
analytics for AI coding-agent token spend, attributed to the projects it was
spent on and measured against the value those projects produced.

This package contains no logic of its own; it exists only so both `npx token-roi`
and `npx project-roi` work. See the `token-roi` README for documentation.
